An easy way to remove blank form fields label on contact form 7 email.

== Description ==

This is an early version of the remove blank form fields label on contact form 7 email for WordPress, please let me know at http://plugins.wrpsolution.com/ if you have suggestion how to make it better.

The plugin is addon for contact form 7. so this plugin activation require contact form 7 plugin.

This plugin is removed blank form field label for contact form 7 email. 	

= Settings For CF7 = 
1. Go to Contact Menu & Select Contact Forms
2. Select Any One Form
3. Here you have add start( [remove] ) and end ( [/remove] ) shortcode inside contact form email setting
4. Ex. if [first_name] value is blank so remove 'First name' label. Follow below code **[remove] First Name : [first_name] [/remove]** *This example is remove 'First Name : ' label for email if value is blank.*




This plugin is maintained by Naresh Goyani, http://plugins.wrpsolution.com/

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/empty-field-label-remover-cf7/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go To Contact Form 7 Menu & Select 'Empty Fields Remover For CF7'
4. Here you have click enable checkbox and click to 'Save Setting'.
